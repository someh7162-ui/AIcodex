================================================================================
                          文件说明
================================================================================

start-server.bat 功能:
  1. 检查 Node.js 是否已安装
  2. 检查账户文件是否存在
  3. 启动 Web 服务器
  4. 显示访问地址

check-quota.bat 功能:
  1. 检查 Node.js 是否已安装
  2. 检查账户文件是否存在
  3. 在命令行显示配额信息

================================================================================
                          常见提示
================================================================================

如果看到:
  "[ERROR] Node.js not found!"
  → 安装 Node.js: https://nodejs.org/

如果看到:
  "[WARNING] Account file not found!"
  → 运行命令: opencode auth login

如果看到:
  "Server stopped"
  → 这是正常的，表示服务器已停止

================================================================================
                        手动运行方法

  cd F:\AI编程\配额使用
  node antigravity-quota-checker.js
