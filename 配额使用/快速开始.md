# 🚀 快速开始指南

## Windows 用户（最简单）

### 方法 1: 命令行查看（推荐新手）

1. **双击** `check-quota.bat`
2. 查看配额信息
3. 完成！

### 方法 2: Web 界面查看（推荐日常使用）

1. **双击** `start-server.bat`
2. 等待服务器启动（看到 "服务器已启动" 消息）
3. 打开浏览器，访问显示的地址（通常是 `http://localhost:8080`）
4. 查看漂亮的配额仪表板
5. 按 `Ctrl+C` 停止服务器

## 常见问题

### ❓ 提示 "未找到 Node.js"

**解决方案：**
1. 下载 Node.js：https://nodejs.org/
2. 选择 LTS 版本（左边的按钮）
3. 安装完成后重启电脑
4. 重新运行

### ❓ 提示 "未找到账户文件"

**解决方案：**
1. 打开命令提示符（CMD）
2. 运行：`opencode auth login`
3. 按照提示登录 Google 账户
4. 重新运行配额检查

### ❓ 端口被占用

**解决方案：**
1. 关闭其他可能使用 8080 端口的程序
2. 或者修改 `antigravity-server.js` 中的端口号：
   ```javascript
   const PORT = 8081; // 改成其他端口
   ```

### ❓ 浏览器打不开

**解决方案：**
1. 确认服务器已启动（看到 "服务器已启动" 消息）
2. 手动在浏览器输入：`http://localhost:8080`
3. 如果还是不行，检查防火墙设置

## 查看配额的意义

### 配额百分比说明

- **绿色（> 30%）**：配额充足，放心使用
- **橙色（10-30%）**：配额警告，建议节约使用
- **红色（≤ 10%）**：配额严重不足，即将限流

### 重置时间说明

显示配额何时恢复到 100%。例如：
- "重置于 2小时 30分钟" = 2.5 小时后配额恢复

### 账户状态

- **可用账户**：配额充足，可以正常使用
- **受限账户**：配额不足，建议等待恢复或切换账户
- **已禁用账户**：在配置中被禁用的账户

## 使用技巧

### 1. 设置自动刷新

Web 界面已自动每分钟刷新一次，无需手动操作。

### 2. 多账户管理

如果有多个 Google 账户：
1. 运行 `opencode auth login` 添加更多账户
2. 配额工具会显示所有账户的状态
3. OpenCode 会自动在账户间轮换

### 3. 查看历史

命令行工具每次运行都会显示实时数据。
如需保存历史记录，可以：
```bash
node antigravity-quota-checker.js > quota-log.txt
```
