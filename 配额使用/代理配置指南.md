## 🚀 快速配置（推荐）

### 方法 1：自动配置（最简单）

1. **双击运行**：`setup-proxy.bat`

2. **输入代理地址**，例如：
   ```
   常见代理格式：
   - 公司代理：proxy.company.com:8080
   - Clash：127.0.0.1:7890
   - V2Ray：127.0.0.1:10809
   - SSR：127.0.0.1:1080
   ```

3. **配置完成后，使用带代理的脚本**：
   - 查看配额：双击 `check-quota-with-proxy.bat`
   - Web 界面：双击 `start-server-with-proxy.bat`

## 📝 手动配置

### 方法 2：临时设置（单次有效）

打开 CMD，每次使用前运行：

```batch
# 设置代理（替换为你的代理地址）
set HTTP_PROXY=http://127.0.0.1:7890
set HTTPS_PROXY=http://127.0.0.1:7890

# 然后运行
node antigravity-quota-checker.js
```

### 方法 3：创建自定义脚本

创建文件 `my-proxy.bat`：

```batch
@echo off
REM 修改这里的代理地址
set HTTP_PROXY=http://127.0.0.1:7890
set HTTPS_PROXY=http://127.0.0.1:7890

REM 运行配额检查
node antigravity-quota-checker.js
pause
```

## 🔍 如何找到你的代理地址？

### Clash 用户
1. 打开 Clash
2. 查看端口号（通常是 `7890`）
3. 代理地址：`127.0.0.1:7890`

### V2Ray / V2RayN 用户
1. 打开 V2Ray
2. 查看 HTTP 代理端口（通常是 `10809`）
3. 代理地址：`127.0.0.1:10809`

### SSR 用户
1. 打开 SSR
2. 查看本地端口（通常是 `1080`）
3. 代理地址：`127.0.0.1:1080`

### 公司代理
1. 打开 Windows 设置
2. 网络和 Internet → 代理
3. 查看代理服务器地址和端口
4. 例如：`proxy.company.com:8080`

## ⚙️ 高级选项

### 排除本地连接（可选）

如果需要排除某些地址不走代理，编辑 `proxy-config.bat`：

```batch
@echo off
set HTTP_PROXY=http://127.0.0.1:7890
set HTTPS_PROXY=http://127.0.0.1:7890
set NO_PROXY=localhost,127.0.0.1,.local
```

### 验证代理是否工作

运行 `test-fetch.bat` 查看是否能连接到 Google API。

## 🆘 还有问题？

### 问题 1：配置后仍然无法连接

**检查清单：**
- [ ] 代理软件是否正在运行？
- [ ] 代理端口是否正确？
- [ ] 代理软件是否开启了"允许来自局域网的连接"？

### 问题 2：如何修改代理配置？

**方法 1：** 重新运行 `setup-proxy.bat`

**方法 2：** 直接编辑 `proxy-config.bat`：
```batch
set HTTP_PROXY=http://新的代理地址:端口
set HTTPS_PROXY=http://新的代理地址:端口
```

### 问题 3：如何临时禁用代理？

使用不带代理的原始脚本：
- `check-quota.bat`（不使用代理）
- `start-server.bat`（不使用代理）
