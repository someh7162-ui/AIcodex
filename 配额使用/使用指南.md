## ❓ 常见问题

### Q: 提示 "[ERROR] Node.js not found!"
**A:** 需要先安装 Node.js
1. 访问: https://nodejs.org/
2. 下载 LTS 版本（推荐）
3. 安装完成后重启电脑
4. 重新运行批处理文件

### Q: 提示 "[WARNING] Account file not found!"
**A:** 需要先登录 Google 账户
1. 打开命令提示符（CMD）
2. 运行: `opencode auth login`
3. 按照提示登录
4. 重新运行批处理文件

### Q: 端口被占用怎么办？
**A:** 修改端口号
1. 打开命令提示符（CMD）
2. 运行:
   ```
   cd F:\AI编程\配额使用
   set PORT=8081
   node antigravity-server.js
   ```

### Q: 如何停止服务器？
**A:** 按 `Ctrl + C` 键

## 💡 使用技巧

1. **推荐使用 Web 界面**
   - 界面更美观
   - 自动刷新（每分钟）
   - 彩色进度条
   - 可以同时查看多个账户

2. **创建桌面快捷方式**
   - 右键 `start-server.bat` → 发送到 → 桌面快捷方式
   - 方便随时查看配额

3. **多账户管理**
   - 运行 `opencode auth login` 可以添加更多账户
   - 工具会显示所有账户的配额状态

4. **配额颜色说明**
   - 🟢 绿色：配额充足 (> 30%)
   - 🟠 橙色：配额警告 (10-30%)
   - 🔴 红色：配额严重不足 (≤ 10%)
