# 📚 我的课表 - 使用说明

## 简介
这是一个简单易用的课表管理网页应用，帮助你轻松管理学期课程安排，并支持多种格式导出。

## 功能特点

### ✨ 核心功能
- 📝 **添加课程**：轻松录入课程信息，包括课程名称、教师、地点、时间等
- 📊 **课表展示**：直观的表格式课表，一周课程一目了然
- ✏️ **编辑课程**：点击课程卡片可查看详情并编辑
- 🗑️ **删除课程**：支持单个删除或清空全部课程
- 💾 **本地存储**：数据保存在浏览器本地，不会丢失

### 📤 导出功能
- **导出Excel** (.csv格式)：可用Excel、WPS等软件打开编辑
- **导出日历** (.ics格式)：可导入到手机日历、Outlook、Google日历等
- **打印课表**：直接打印或保存为PDF

## 使用步骤

### 第一步：设置学期开始日期
1. 在页面底部的"学期设置"区域
2. 选择你的学期第一周周一的日期
3. 点击"保存学期设置"
4. 这个日期用于导出日历文件时计算每节课的具体日期

### 第二步：添加课程
1. 点击"➕ 添加课程"按钮
2. 填写课程信息：
   - **课程名称**：如"高等数学"
   - **任课教师**：如"张老师"
   - **上课地点**：如"教学楼A101"
   - **星期**：选择周一到周日
   - **节次**：选择上课的时间段
   - **开始周/结束周**：课程的周次范围（如第1周到第18周）
   - **周次说明**（可选）：
     - 留空表示每周都上
     - 填"单周"表示单周上课
     - 填"双周"表示双周上课
     - 填"1,3,5"表示第1、3、5周上课
     - 填"1-5,8-10"表示第1-5周和第8-10周上课
3. 点击"保存"

### 第三步：查看和管理课程
- **查看详情**：点击课表中的课程卡片查看完整信息
- **编辑课程**：在详情页点击"编辑"按钮
- **删除课程**：在详情页点击"删除"按钮

### 第四步：导出课表

#### 导出Excel表格
1. 点击"📊 导出Excel"按钮
2. 下载的.csv文件可用Excel打开
3. 包含所有课程的详细信息

#### 导出日历文件
1. 确保已设置学期开始日期
2. 点击"📅 导出日历"按钮
3. 下载的.ics文件可导入到：
   - iPhone/iPad：用文件应用打开，添加到日历
   - Android：用日历应用导入
   - Outlook：文件 → 导入/导出 → 导入iCalendar文件
   - Google日历：设置 → 导入和导出 → 从计算机中选择文件

#### 打印课表
1. 点击"🖨️ 打印课表"按钮
2. 在打印预览中可以：
   - 直接打印到纸张
   - 或选择"另存为PDF"保存电子版

## 时间段说明
程序默认设置了5个常用时间段，你可以根据需要修改：

- **第1-2节**：08:00-09:40（上午第1大节）
- **第3-4节**：10:00-11:40（上午第2大节）
- **第5-6节**：14:00-15:40（下午第1大节）
- **第7-8节**：16:00-17:40（下午第2大节）
- **第9-10节**：19:00-20:40（晚上）

如果学校时间不同，可以修改 `script.js` 文件中的 `timeSlots` 配置。

## 快捷键
- **Ctrl+N** (或 Cmd+N)：快速打开添加课程窗口
- **ESC**：关闭弹窗

## 注意事项
1. **数据存储**：课表数据保存在浏览器本地存储中
   - 清除浏览器缓存会删除课表数据
   - 建议定期导出备份
   - 更换浏览器或电脑需要重新录入

2. **浏览器兼容性**：
   - 推荐使用Chrome、Edge、Firefox等现代浏览器
   - 需要启用JavaScript

3. **导出日历**：
   - 导出日历前必须设置学期开始日期
   - 确保日期设置正确，否则导入后时间会错乱

## 技术支持
如果遇到问题或有改进建议，可以：
- 检查浏览器控制台是否有错误信息
- 确认浏览器允许JavaScript运行
- 尝试刷新页面或清除缓存后重试
